APIs available for pulling police shootings and police brutality

/shootings will return all records in our DB

/shootings/{id} will return just record number id

/shootings/{id1}-{id2} will return records from id1 to id2

/shootings/last20 will return the last (most recent) 20 records

/count will return the count of all shootings and brutality records

/count/state/name will return the count of all shootings and brutality records by state

/count/state/abbv will return the count of all shootings and brutality records by state abbreviation

/count/shootings will return the count of all shootings records

/count/shootings/state/name will return the count of all shootings records by state

/count/shootings/state/abbv will return the count of all shootings records by state abbreviation

/count/brutality will return the count of all brutality records

/count/brutality/state/name will return the count of all brutality records by state

/count/brutality/state/abbv will return the count of all brutality records by state abbreviation

Future functionality will also include police brutality data
